# UPS-Hysteresis-IoT
Uninterruptible Power Supply with Voltage Hysteresis Control for Low Power Networking Devices with IoT Based Monitoring

ProvEnergProcess_New_2026_V5.ino (Controller code for ATmega328P)
UPS_WIL_Data_Summary_2_en_tmp.ipynb (Results of monitoring 2)
UPS_WIL_Data_Summary_en.ipynb (Results of monitoring 1)
WiFiWebClientRepeatingMyServerQuery_VGVRVBVI.ino (Monitoring code for R7FA4M1AB3CFM#AA0)

This research proposes an uninterruptible power supply with voltage hysteresis control for low power networking devices, the system is based on the ATmega328P microcontroller for control and R7FA4M1AB3CFM#AA0 microcontroller for monitoring, low-power networking devices such as access points, switches, routers, and others require an uninterrupted power source in case of a power outage. The system switches a relay circuit immediately upon detecting a power failure in the electrical grid and prevents the networking devices from having to shut down or restart. Voltage changes occur when the output circuit switches. The controller uses voltage hysteresis that refers to the difference in voltage levels that cause a circuit to change state depending on whether the voltage is rising or falling, this means that a circuit's output does not instantly change to the input voltage, hysteresis helps prevent rapid and unwanted changes in a circuit's output. The ATmega328P microcontroller can potentially achieve a performance of 20 Million Instructions Per Second,  the switching process is so fast that the power transfer for the load does not allow the networking device to enter a power loss reboot state or be power off for a few seconds. The monitoring system is reading the variables signals of the controller and is responsible for sending the data to a server. This system follows the IoT model, this helps to improve the system and review the performance.
